# Changelog

All notable changes to RemoteCraft are documented here.

## [0.2.1] - 2026-07-17

### Added

- Guided Linux installer with SSH fingerprint confirmation and remote requirement checks.
- User-level systemd service with loopback binding and conservative hardening.
- Animated workflow demo and repository social-preview asset.
- Public roadmap, issue forms, and contributor-friendly starter tasks.

### Changed

- Reworked the quick start around the guided installer while preserving manual setup.
- Made generated configuration safe for both systemd and direct manual startup.
- Improved repository discovery metadata and launch documentation.

## [0.2.0] - 2026-07-17

### Added

- Installable FastAPI control plane and responsive web dashboard.
- Bearer authentication and browser session-only token storage.
- Strict SSH host-key verification and validated remote command construction.
- Verified Vanilla server downloads using Mojang release metadata and SHA-1 checksums.
- Server lifecycle, console, log, and safe deletion operations.
- Atomic JSON metadata storage, 45 focused tests, packaging, CI, and security guidance.
